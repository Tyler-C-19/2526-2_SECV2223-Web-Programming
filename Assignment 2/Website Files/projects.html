Student Feedback and Complaint System

Programming Technique II • 2024/2025

Built using fundamental C++ concepts learned throughout the Programming Technique II course, the Student Feedback and Complaint System is a simple, yet feature-rich system that uses the command line interface (CLI) to process feedback and complaints. It includes advanced features such as a logging in feature, dual-user mode (student and admin) as well as a file-based database system to store all reports.

InfinitePROTECH

Technology & Information System • 2024/2025

InfinitePROTECH is a proposed mobile application designed to promote cybersecurity actively, through features such as fraud detection and passively through education. One stand-out feature would be 'Lockdown Mode', which is a last-resort protection method to protect users' devices. For this project, we created a low-fidelity prototype to showcase how the app would function and be seen by users.
